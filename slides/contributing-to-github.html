The Master Reference

AIMS : - Note that MR is currently undergoing a huge overhaul - if you have issues with the structure of the course, the master reference is the place to create an issue concerning it
- To define a methodology for building a curriculum.
- To supply a set of resources that can be used to build a curriculum.
- To allow a single point of collaboration between campuses without impeding on the autonomy of any one campus
INCLUDES: - course book
- contributing guidelines
- current maintainers
- code of conduct
- list of gitter channels
- pastoral care walk guidelines

London Programme repo

- A place for London specific schedules
- A repository for coordinating course activities and engaging the wider membership of the CIC.
- current cohort info
- current schedule
- minutes of business meetings
- SGC notes
- most 'general' FAC info

Members repo

- register of current members
- bye-laws
- complaints policy
- membership charter
- list of decisions to be put to members

HQ repo

- objectives, mission, vision, history, core values, annual reports, branding, jobs, contracts, finances etc.
