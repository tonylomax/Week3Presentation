Why are we learning this?

  • Pretty much all the data used by most applications is sent via API's.

  • CityMapper uses the TFL API, Timeout uses the CityMapper API etc...

  • Huge amounts of data can be accessed easily, you only request data when you need it and it usually comes in a familiar format - as a Javascript object.
